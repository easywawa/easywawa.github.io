### 故障描述：

```
XFS: Filesystem has duplicate UUID – can’t mount
```

### 解决方案:

##### 1、临时解决

使用mount命令使用nouuid选项进行挂载 （-o 指定参数）

```
mount -o rw,nouuid /dev/sda3 /mnt
```

重启启动后将失效

##### 2、永久解决  [对于xfs文件系统]

**使用xfs_admin工具为此分区生成新的UUID**  (-U  指定uuid)

```
uuidgen ##生成uuid

xfs_admin -U <uuid> /dev/sda3
```
