# 命令

```
Ctrl + a ：移到命令行首

Ctrl + e ：移到命令行尾



Alt + f ：按单词前移，光标后移（右向）  front

Alt + b ：按单词后移，光标前移（左向） backend



Ctrl + u ：从光标处删除至命令行首

Ctrl + k ：从光标处删除至命令行尾

Ctrl + w ：从光标处删除至字首

Alt + d ：从光标处删除至字尾
```
